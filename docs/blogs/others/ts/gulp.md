## 安装依赖

- 安装全局依赖

```bash
npm install gulp-cli -g
```

- 安装开发依赖

```bash
npm install typescript gulp gulp-typescript -D
```

## 创建`gulpfile.js`

在项目的根目录创建`gulpfile.js`

<<< @/docs/blogs/others/ts/gulpfile.js

:::tip 测试

在控制台输入：

```bash
gulp    // 执行`gulpfile.js`
node folder/file.js     // 运行编译完的具体js文件
```

:::

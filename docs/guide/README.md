## Quick Star

1. 安装

    ```bash
    npm install
    ```

2. 启动本地服务器（自动打开浏览器进行本地预览）

    ```bash
    npm run docs:dev
    ```

3. 发布运行项目根目录的`deploy.sh`进行部署
   ::: tip
    - 自动构建
    - 自动提交到`gh-pages`分支
    - 可以自定义`deploy.sh`中的配置
   :::

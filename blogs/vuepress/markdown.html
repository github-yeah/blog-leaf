代码高亮行数区间规则 逗号之间不要有空格

  • 行数是从1开始（不是从0开始）
  • 单行: {n}
  • 跳行: {n1,n3,n5}
  • 连续多行: {n1-n3,n5-n9}
  • 组合: {n1,n3-n5,n7,n8-n9}

# 代码导入

语法规则：

<<< @/代码文件路径#region{高亮行数区间}
1

TIP

  • 此处@/ 默认指的是项目根目录
  • 高亮行数区间参照代码块以及语法高亮
  • #region 是可选项，可以实现导入代码文件中的局部代码块
    • 在代码文件中通过#region xxx指定截取的代码块开始区域
    • 在代码文件中通过#endregion xxx指定截取的代码块结束区域
    • 在导入代码时通过<<< @/代码文件路径#xxx来指定设置的区域

通过#region导入文件的部分代码块

文件路径：blog-leaf/src/vuepress/vuepress-code-snippet.js

输入：

<<< @/src/vuepress/vuepress-code-snippet.js#snippet1{2}
1

输出：


 

import: function () {
    console.log("本行高亮!");
}
1
2

# 直接使用原生JS和CSS

<!-- 在md文件中输入以下内容 -->

<!-- 样式 -->
<style>
    .box {
        background: yellow;
        text-align: center;
    }
</style>

<!-- html -->
<div id="mydiv">
    我是一个div, id="mydiv", 通过js设置样式
</div>

<div class="box">
    我是一个div, class="box"
</div>

<!-- js -->
<script>
    window.onload = function () {
        var mydiv = document.getElementById("mydiv");
        mydiv.className = "box";
    }
</script>
